# Smart Undo-Redo Notes App using Stack

## 📌 Description
This is a mini project developed in C language that demonstrates the use of Stack Data Structure.

## 🚀 Features
- Add Notes
- View Notes
- Undo Last Note
- Redo Last Note

## 🧠 Data Structure Used
Stack (LIFO – Last In First Out)

## ⚙️ How It Works
Two stacks are used:
1. Notes Stack – stores current notes
2. Redo Stack – stores undone notes

Undo moves note from Notes Stack to Redo Stack.
Redo moves note back to Notes Stack.
